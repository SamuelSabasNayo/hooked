## To get started

### Prerequisites

Make sure you have NodeJS installed on your computer by entering  `node -v ` on your terminal.

### Installation

Clone the app
* ```https://github.com/atlp-rwanda/space-bn-frontend.git```

Install all the packages
* ```npm install ```

Run the server
*  ```npm start ```

## Testing
Run Test case
* ```npm run test```
